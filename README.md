# Hangul.js

> Hangul.js는 한글로 이루어진 문장의 자음과 모음을 분리하는 자바스크립트 라이브러리입니다. 자모 분리 또는 초성 검색에 사용할 수 있습니다.

## 설치 및 사용방법

### node.js

```bash
npm install hangul-js
```

```js
const Hangul = require("hangul-js");
```

### 일반 웹 페이지

```html
<script src="https://unpkg.com/typingchanger" type="text/javascript"></script>
```
